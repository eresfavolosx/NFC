## 2024-05-18 - Keyboard Navigation & Missing ARIA Labels\n**Learning:** Action containers that are visually hidden until hover (like `.link-card-actions`) must use `:focus-within` alongside `:hover`. Otherwise, they become invisible and inaccessible to keyboard users navigating via Tab. Additionally, icon-only buttons require explicit `aria-label` attributes for screen reader accessibility.\n**Action:** Always verify keyboard navigation for UI components hidden by default and ensure all non-text interactive elements have appropriate ARIA labels.
